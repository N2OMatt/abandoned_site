About the title...

Let's let the small talk before us and move for what really matters

I have a site... and you probably now that since you're on it :D

My site and the www.amazingcow.com is so called static generated. By saying this, I mean that I have no PHP or database involved at all.
The entire content is generated offline by a set of shell and python scripts on my machine. I choose do this way because it appeared to be more reasonable to me at time.

Yes, I tried a couple of times make a blog, have a site and all this stuff.

The reasons...

... that led me to think that a couple of shell scripts would surfice was:

  • I have a huge amount of certifications that I want to show in site.
  • The doxygen documentation for the projects that I made shall be include on the content.
  • All the photos from places that I visited should be grouped together.
But the main reason was that I want that everything to be built automatically.
So, if I create another project, or take another certification, all (in theory) that I need to do is, put the content on the directory and rebuild the site.

What I built:

Certifications

Tackling the first issue, i.e. the certifications. I had them on a github repo.
They're almost on pdf format, but I need to display png. This is really easy, I just write some shell scripts to use Image Magick and convert the images.
Next, more shell script to generate a default html content with the certification png and the it's title, year, etc.

Pretty easy... and worked fine ;D

Doxygen Documentation

Next I need handle the project's doxygen documentation.
That was even easier. All that I need to do is:

  • Clone the project
  • Go to it's directory
  • Run doxygen
Seriously, that's all ;D

I get the list of the projects using the Github API, so isn't that hard to iterate for all the projects and stuff.

Again, pretty easy and really worked fine ;D

Photos

Lastly, I want that all my photos be automatically handled by the scripts.
With this I mean, I put the photos, the event and the site builder generate all the pages, links, everything that is needed.

Yep, this was just a desire... I even didn't start to implement this feature.
