Engines

  • MonsterFramework
    • Tendo como base o Cocos2d-x v3.0, esta “micro engine” facilita várias tarefas comuns no desenvolvimento de jogos.
    O foco é facilitar ainda mais o Cocos2d-x como funções mais concisas e um API mais clara.
    Tarefas como enviar um “tweet”, e-mail ou interfacear com o dispositivo também são providas de uma maneira mais alto nível, tornando assim o código mais portável.
  • Lore
    • Framework criado utilizando SDL2, em constante desenvolvimento, facilita o desenvolvimento de jogos e outras aplicações interativas.
    Traz muitas ideias de outros frameworks e libs como XNA, Starling, Flixel entre outros.
    Escrito em C++.
