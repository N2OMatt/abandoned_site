Social Bus, Projeto de Conclusão de Curso

Prestei “mentoring” técnico e organizacional a equipe do Eduardo Pimenta no trabalho de conclusão de curso de técnico em informática no COTEMIG.

A solução foi desenvolvida utilizando um “front-end” em C# e Silverlight para Windows Phone com o “back-end” sendo produzido em PHP e MySQL.

Feira Hippie, Projeto de Conclusão de Curso

Projeto produzido para Windows Phone utilizando C# e Silverlight no “front-end”.

A solução requeria a disponibilização de mapas para o usuário final, para isso foi utilizado a API do BingMaps provida pela Microsoft.

A equipe era formada de apenas uma pessoa portanto além do “mentoring” técnico e organizacional bastante codificação foi precisa.
